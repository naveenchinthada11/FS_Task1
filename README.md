# FS_Task1
This is a modern full-stack portfolio website built using HTML, CSS, JavaScript, PHP, and MySQL. It showcases my projects, skills, and contact information with a clean and responsive user interface.
This project is a modern full-stack portfolio website developed to showcase my skills, projects, and web development knowledge. It is built using core web technologies including HTML5, CSS3, JavaScript, PHP, and MySQL.

The portfolio features a responsive and user-friendly interface with a professional design. It includes dynamic elements such as GitHub project integration using the GitHub API, a dark/light theme toggle, and a fully functional contact form connected to a MySQL database.

This project demonstrates both frontend and backend development skills and reflects real-world web application structure and functionality.

🚀 Key Highlights
Responsive and modern UI design
Dark/Light mode toggle with persistent settings
Dynamic GitHub repositories display
Contact form with PHP and MySQL integration
Form validation using JavaScript
Resume download feature
Organized project structure
🎯 Objective

The main goal of this project is to build a complete portfolio that demonstrates:

Frontend design and responsiveness
Backend data handling with PHP
Database integration with MySQL
Use of APIs (GitHub API)
Deployment using GitHub
